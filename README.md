# Scelta abbonamento
Sceglie i migliori abbonamenti per farti risparmiare
Per la modifica dei prezzi degli abbonamenti vanno modificate le prime righe del file python e stampa il prezzo finale e una lista di giorno - abbonamento per sapere quale abbonamento usare e quando

Avviato il programma va inserito un . per ogni giorno in cui non si viaggia e un carattere quando invece ci si sposta con i mezzi.
